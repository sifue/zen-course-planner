# ZEN大学 カリキュラム・ツリー構造化データ - インデックス

このフォルダには、以下の3つのカリキュラム・ツリーを、LLMやプログラム向けに構造化した Markdown を格納している。

## 対象ファイル
- `zen_curriculum_tree_economy_markets_structured.md`
- `zen_curriculum_tree_data_science_structured.md`
- `zen_curriculum_tree_information_structured.md`

## 共通方針
- PDF の接続線は、原則として **推奨履修順・学習上の接続** として解釈する。
- **正式な前提科目** はシラバスを優先する。
- 図で複数の線が合流・分岐して個別対応が切り分けられない場合は、
  - `group_to_group`
  - `group_to_one`
  - `broad_feeder_relation`
  のような表現で保持する。
- 学年・クオーターの配置は、図上の位置を保持する目的で記述している。
- 単位数、必修/選択、正式な開講学期は別の科目マスタで管理するのが望ましい。

## 推奨運用
- この Markdown は、**検索・要約・経路推定・履修相談の補助**に使う。
- 単位判定や厳密な要件判定には、別途作成した卒業要件データおよび科目マスタを使う。
