## 本文

## 概要

完全な未経験者向けのプログラミング入門講座。HTML、CSS、JavaScript を身につけ、Webページ制作ができるようになる。JavaScriptというプログラミング言語を重点的に学修し、思い通りの動きをWebページにつけることができるようになり、CSS を使ってWebページのデザインを自由に変えられるようになる。ソーシャルコーディングサービスのアカウントを作成し、作成したWebページを公開する。
学修にはPCが必要である。

## 到達目標

JavaScriptとHTML/CSSを通じて、Webアプリケーション開発の基礎技術を修得し、独自のWebサービスが開発できる実践力を身につける。ブラウザやエディタやソーシャルコーディングサービスなどの開発環境にも触れ、デジタルツールとしての範疇を超え、問題解決のツールとして使用できる力を身につける。さらに作成したWebページを公開する実践を通じ、ITツールを使って自身の表現力を鍛える。プログラミングを使った開発経験を積み、デジタルツールを用いて情報収集、表現、コミュニケーションする力を獲得することを目標にする。

## 講義計画

| 回 | タイトル | 内容 |
| --- | --- | --- |
| 1 | プログラミング体験をしてみよう | プログラミングの基礎体験を通じ、その楽しさと基本的な考え方を身につける。さらにHTMLの基本的な扱い方も修得し、プログラミングの入門としての理解を深める。 |
| 2 | さまざまなHTMLタグ | さまざまなHTMLタグ①から⑥までを学び、各タグの特性と使用方法を修得する。Webフロントエンド開発に必要な基本的知識を身につける。 |
| 3 | Webページ制作とJavaScriptをはじめよう | 身に付けたHTMLを応用して自己紹介ページの作成スキルを身につける。JavaScriptを体験する。 |
| 4 | JavaScriptでの計算 | JavaScriptを用いた計算の技術をマスターする。各トピックを通じて、コード記述の基礎から応用までを身につけ、問題解決のスキルを磨く。 |
| 5 | JavaScriptで論理を扱う | JavaScriptを使用して論理を扱う方法を学び、プログラミングスキルを磨く。特に、論理を扱う様々な手法についての理解を深めることを目指す。 |
| 6 | JavaScriptのループとコレクション | JavaScriptのループとコレクションについて理解を深め、その活用方法を学ぶ。具体的には、ループの基本的な使い方から応用まで、また、配列の使い方も含めて総合的に学修する。 |
| 7 | JavaScriptの関数 | JavaScriptの関数の基礎から応用までを徹底的に学び、より高度なプログラミング技術の修得を目指す。関数の定義方法や使い方を修得する。 |
| 8 | JavaScriptのオブジェクト | JavaScriptのオブジェクトを修得し使えるようになる。また、ここまで身に付けた JavaScript を応用して簡単なゲーム制作を体験する。 |
| 9 | CSS | CSSの基本から一歩進んだプログラミング技術までを修得する。CSSの基本を固め、CSSを使ったプログラミングを通し実践力を身につける。 |
| 10 | Webページの企画とデザイン | Webページの企画からデザインまで一連の流れを学修する。具体的な企画立案からデザインの基礎、応用までを段階的に理解し、自分のサイトを作り上げる力を身につける。 |
| 11 | いいところ診断機能の開発 | 診断機能の作成を通じて開発の一連の手法を学び、自身で診断機能を作成する能力を身につける。各段階における技術や知識の理解を深める。 |
| 12 | いいところ診断機能の組込み | 動きのあるWebページの作成方法を身につける。JavaScirptでDOMを扱う技術を修得する。 |
| 13 | ツイート機能の開発とソーシャルコーディングサービスでの公開 | WebページにSNS共有を行うボタンを配置する方法を身につける。また、ソーシャルコーディングサービスを使ったウェブサイトの公開手順についても学び、最終的に自作の機能を公開する能力を身につける。 |
| 14 | CSSフレームワークによるデザインの改良 | CSSフレームワークを用いたデザイン改良を学び、各段階毎の学修を通じて実践的なスキルを修得する。最終的には独自のデザインを作成できる能力を身につける。 |
| 15 | ポートフォリオサイトを作ろう | ポートフォリオサイトの制作技術を学び、自己紹介や作品集などを一つのサイトにまとめる技術を身につける。具体的な課題に取り組むことで、実践的なスキルを修得する。 |
