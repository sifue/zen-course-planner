## 概要

急速にグローバル化が進む現代社会の中で活躍するために、国境を越えた国々、文化、世界観についての知識が不可欠である。本授業では、海外で読まれた具体的な日本文学作品を取り上げ、比較考察の方法を通して、文化の流通と変容の詳細な過程について考えたい。

## 到達目標

本授業では、日本文学は海外でいかに受容され、どのように評価されてきたのかについて知識を得ながら、日本文学の代表的な作品について知識を得ることができる。なお、文化生成の過程について理解することができ、文学作品や評論の読解力が身につき、思考力、表現力も育成することができる。

## 講義計画

| 回 | タイトル | 内容 |
| --- | --- | --- |
| 1 | オリエンテーション | 本科目の概念、利用する資料・テキスト、評価基準及び学び方などを説明し、本科目で何を目指すのか詳しく説明する。 |
| 2 | 国文学と世界文学 | 「国文学」と「世界文学」を定義し、世界のなか日本文学を位置づけるとはいかなることかについて考える。 |
| 3 | 作られたカノン：世界文学としての『源氏物語』① | 19世紀末・20世紀初頭の国内外における『源氏物語』の受容と評価について考える。 |
| 4 | 作られたカノン：世界文学としての『源氏物語』② | 1920年代に行われた『源氏物語』の英訳を取り上げ、それが国内における『源氏物語』の再評価にどのように影響を与えたのかについて考える。 |
| 5 | 作られたカノン：世界文学としての『源氏物語』③ | 戦後～現在にかけて『源氏物語』はいかにしてカノン化されてきたのかについて考える。 |
| 6 | 織物としてのテキスト：世界文学としての『方丈記』① | 国内における『方丈記』の受容について踏まえた上で、海外におけるその受容の特徴について考える。 |
| 7 | 織物としてのテキスト：世界文学としての『方丈記』② | 『方丈記』の代表的な英訳を取り上げ、本作品を世界文学として位置付ける。 |
| 8 | 中間発表 | 各グループが決めたテーマについて発表を行う。 |
| 9 | 中間発表 | 各グループが決めたテーマについて発表を行う。 |
| 10 | 非対称性と文学：夏目漱石と世界文学 ① | 19世紀末・20世紀初頭の時代空間において、漱石の問い「文学とは何か」について考える。 |
| 11 | 非対称性と文学：夏目漱石と世界文学 ② | 漱石の文学を世界の文学の視点で考える。 |
| 12 | 世界文学としての俳句 ① | 20世紀初頭に俳句はどのように海外に紹介され、世界各地でいかに受容されたのかについて考える。 |
| 13 | 世界文学としての俳句 ② | 戦後における俳句ブームを取り上げ、世界文学としての俳句とはいかなるものかについて考える。 |
| 14 | 世界文学としての日本現代文学 ① | 海外で読まれている日本の現代文学作品を取り上げ、現代における世界文学とは何かについて考える。 |
| 15 | 世界文学としての日本現代文学 ② | 海外で読まれている日本の現代文学作品を取り上げ、現代における世界文学とは何かについて考える。 |
