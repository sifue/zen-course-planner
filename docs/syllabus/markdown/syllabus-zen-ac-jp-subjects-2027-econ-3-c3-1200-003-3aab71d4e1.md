## 本文

## 概要

経済や経営で使われる統計学の基礎知識やさまざまな統計手法を理解するために不可欠となる理論とその応用について、「基礎編」と「実践編」の二段階に分け指導する。演習では、日常生活において観測できる興味深く具体的なデータを豊富に活用し、統計学の基礎と実践を結びつける訓練を行う。「基礎編」では記述統計、確率と統計の関係などを扱い、「実践編」では確率モデル、時系列データと時系列グラフなどを扱う。

## 到達目標

多様なデータの統計的分析を通じて、社会における現象の傾向やパターンを発見するスキルを身につけ、客観的なデータ評価や仮説構築、意味の解読に取り組むことで思索力を向上させることを目指す。このような数理的理解は、社会や経済の課題への深い理解と、問題解決のための有益な提案を生む力につながり、学生は将来の課題に真摯に取り組むスキルとマインドの両方を強化することができる。

## 講義計画

| 回 | タイトル | 内容 |
| --- | --- | --- |
| 1 | イントロダクション | さまざまなデータの種類、データの分析・分類、記述統計と推測統計。 |
| 2 | 基礎編：データをまとめよう | 図表による記述統計：度数分布表、ヒストグラム、階級幅。 |
| 3 | 基礎編：データの位置を調べてみよう | さまざまな中心の指標：平均、中央値、最頻度の違いを学ぶ。 |
| 4 | 基礎編：さまざまな平均 | 加重平均と幾何平均の違いと使い方。 |
| 5 | 基礎編：データの散らばり具合を調べてみよう | 平均絶対偏差、分散、標準偏差、変動係数を理解する。 |
| 6 | 基礎編：散らばりがわかるとどんなことがわかる？ | ローレンツ曲線とジニ係数、基準化変数。 |
| 7 | 基礎編：二つのデータの間の関係を探ってみよう | 散布図、共分散、相関係数をもちいて相関関係を確認する。 |
| 8 | 基礎編：データの背後にあるものを探る | 確率と統計の関係、推定と仮説検定を理解する。 |
| 9 | 実践編：データに直線をあてはめてみよう | 単回帰分析と回帰式のあてはまりの評価。 |
| 10 | 実践編：あてはめられた直線の説得力を検証しよう | 確率モデル、係数と定数項の値の変動、係数のt検定、t検定による変数選択。 |
| 11 | 実践編：複数の要因で説明しよう | 重回帰分析とあてはまり尺度、個々の係数のt検定、多重共線性。 |
| 12 | 実践編：曲線をあてはめてみよう | モデルの関数形：対数線形モデル、多項式モデル、双曲線モデルについて学ぶ。 |
| 13 | ダミー変数を使ってみよう | 一時的ダミー変数、定数項ダミー変数、係数ダミー変数を正しく使えるようになる。 |
| 14 | ものごとの変化をとらえる | 時系列データと時系列グラフ、指数化、移動平均。 |
| 15 | 季節の影響を除いたデータを見る | 時系列データの分解、加法的モデルと乗法的モデル、季節変動の対処法。 |
